## Platform-specific features

### iOS only

- **StoreKit 2 support**: Native integration with Apple's latest StoreKit APIs
- **App Store receipt validation**: Automatic receipt verification
- **Family Sharing**: Support for subscription sharing

### Android only

- **Google Play Billing Library 5+**: Latest billing APIs
- **Pending purchases**: Handle deferred payment methods
- **Proration modes**: Subscription upgrade/downgrade handling

### Cross-platform considerations

React Native and Flutter SDKs provide unified APIs but rely on native platform capabilities underneath. Some platform-specific behaviors may differ slightly.

## Version requirements

- **iOS**: iOS 13.0+
- **Android**: Android 5.0+ (API level 21+)
- **React Native**: React Native 0.64+
- **Flutter**: Flutter 2.0+
